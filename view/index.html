Network Graph

What is This!!!

The outermost circles represent 3 groups.

1. Nodes that are similar to the Truth Value

2. Nodes that are not similar to the Truth Value but have been labelled

3. Nodes that are not similar to the Truth Value and have not been labelled


Each of these Circles is further divided into 5 parts based on count of followers as follows:

1. Less than 100 followers

2. Followers between 100 and 500

3. Followers between 500 and 1000

4. Followers between 1000 and 10000

5. Followers greater than 10000


Then again, each of these groups is sub divided into highly connected and lowly connected nodes based on the property that a highly connected node is the one where each node has got retweets greater than 5% of their total followers.
